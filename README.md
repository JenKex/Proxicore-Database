# Proxicore-Database

1: Säkerställ att ni har PHP, Laravel och Composer installerade.
2: Kopiera över den bifogade .env-filen till /Proxicore-Database.
3: Öppna ett terminal-fönster (PowerShell, förslagsvis) i mappen /Proxicore-Database.
4: Kör 'php artisan migrate' och 'composer run dev'.
5: Kontakta mig om ni stöter på några fel.
